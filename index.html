매출 데이터는 민감합니다. 그래서 위젯은 — 오직 보여드리는 일만 합니다. 설치 전에, 어떻게 동작하는지 알고 시작하세요.

아임웹 공식 제공

외부 도구가 아닙니다. 아임웹이 직접 만들고 운영합니다. 대표님 계정과 그대로 연결됩니다.

안전한 인증

아임웹 사이트 관리자 계정으로 연결합니다. 비밀번호나 API 키를 위젯에 직접 입력하지 않습니다.

읽기 전용

매출을 보여드리기만 합니다. 상품 정보, 주문, 회원 데이터는 일절 건드리지 않습니다.

설치 30초

다운로드, 로그인 한 번이면 자동으로 사이트가 연결됩니다. 복잡한 설정도, 카드 등록도 없습니다.

데이터 수집 없음

자동 분석, 진단 업로드, 사용 로그 수집을 하지 않습니다. 위젯은 매출을 보여주는 데 필요한 요청만 수행합니다.

20분마다 자동 업데이트

매출 데이터는 20분마다 자동으로 갱신됩니다. 위젯을 보시면 그 사이의 변화가 늘 반영됩니다.

설치 가이드

다운로드부터 첫 위젯까지,
흐름은 세 단계입니다.

앱을 열고 사이트를 추가하면 끝입니다. macOS는 바탕화면 위젯을 직접 추가하고, Windows는 바탕화면 위젯 창에서 바로 확인합니다.

01

앱 설치

Mac 또는 Windows용 설치 파일을 내려받아 실행합니다. 앱은 첫 실행 때 필요한 실행 레이어를 자동으로 준비합니다.

02

사이트 추가

사이트 관리에서 아임웹 사이트 관리자 계정으로 로그인하고 연결을 허용합니다. 여러 사이트를 차례로 추가할 수 있습니다.

03

위젯 선택

macOS는 바탕화면에서 위젯을 추가합니다. 각 위젯에서는 전체 사이트 또는 사이트 1개를 골라 표시할 수 있습니다.

Mac에 설치하기 Windows에 설치하기

Windows 프리뷰는 업데이트 무결성 서명을 사용하지만, 이번 버전에는 Windows 게시자 서명이 아직 포함되지 않습니다. 설치 중 Microsoft SmartScreen 안내가 보일 수 있습니다.

대표님의 사업,
이제 책상 위에.

설치하고 아임웹 사이트를 한 번 추가하면, 그 다음부터는 위젯이 계속 켜져 있습니다. 대표님의 책상 한 켠에서, 사업의 흐름이 계속 보입니다.
